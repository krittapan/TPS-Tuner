This Privacy Policy explains how TPS Tuner accesses and uses information to provide its core functionality.

1. Information the App Accesses

TPS Tuner may access the following only to provide core functionality:

  • Bluetooth: to scan, connect, and communicate with TPS devices.
  • Location (while in use): required by some Android versions for Bluetooth scanning.
  • Local files: only when users explicitly import or export configuration files.

2. How Information Is Used

Information is used to:

  • connect to supported devices,
  • send and receive tuning configuration data,
  • read and save configuration files selected by users,
  • show app and connection status.

3. Data Collection and Sharing

  • TPS Tuner does not sell personal data.
  • TPS Tuner does not automatically share personal data with third parties.
  • Configuration data is handled locally on user devices and in user-selected files.

4. Permissions

The app requests only permissions needed for functionality:

  • Bluetooth permissions
  • Location permission (when required for BLE scanning)
  • File access permissions (when users choose files)

Users can revoke permissions at any time in system settings.

5. Data Security

Reasonable measures are applied to protect data handled by the app.

6. Children's Privacy

This app is not directed to children under 13 and does not knowingly collect personal data from children.

7. Changes to This Policy

This policy may be updated from time to time. The effective date above will be revised when changes are made.
